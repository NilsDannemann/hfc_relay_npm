# HFC Relay Framework
HFC Relay Framework (SCSS)

<br><hr>
## Installation

#### Via npm
1. Install: ```npm install hfc-relay --save-dev;```
1. Import: ```@import 'node_modules/hfc-relay/src/hfc-relay.scss';```
