# HFC Relay Framework
HFC Relay Framework (SCSS)

<br><hr>
## Installation

#### Via npm
1. Install:<br> 
```npm install hfc-relay --save-dev;```

1. Create _config.scss
```npm install hfc-relay --save-dev;```

1. Import relay & config:<br> 
```
// HFC Relay - Main
@import 'node_modules/hfc-relay/src/hfc-relay.scss';
// HFC Relay - Config Overrides
@import 'config.scss';
```
